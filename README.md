# java-filmorate

Filmorate - a service, that helps you to find the movie you need

1. Find films you need
2. Make friends
3. Give the films ratings

Stack:

Spring Boot, Spring Data JPA, Hibernate, Maven, Lombok, PostgreSQL, Docker, JUnit, MockMvc.
